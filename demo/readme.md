# Demos of numpy-cnn
Here we release some supported demos.

## CRAFT text detector
Demo for scene text detector with model [CRAFT](https://github.com/clovaai/CRAFT-pytorch).
Run ```python main.py``` inside folder ```craft_text_detector```.
